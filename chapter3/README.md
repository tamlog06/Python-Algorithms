# 数え上げ入門
総和について

## 総当たり戦
n人が握手するのは何回か、n個のノードの完全グラフのエッジの数など  
総当たり戦の試合の数は次のようにかける  

    sum(i for i in range(0, n-1)) == n*(n-1)/2  


## 勝ち抜き戦
勝ち抜き戦の試合の数は次のようにかける  

    h = log(n)
    sum(2**i for i in range(0, h-1)) == n-1
    
ここから、完全二分木はn-1の内部ノードを持つことがわかる
